CSS terms you might wanna know

TermUsage
em is a value used to set font-size. 1em is by default 16px font, whereas 0.5em is 8px font. It changes based on the selector.
rem is similar to em but is always set to the default of the html, usually 16px.
border-collapse:
collapse		 removes the borders between different cells in a table. It makes the tables look nice and homogenous!
classes are a way to set styling for multiple things. They can be declared like so: .colortable, which would declare class colortable. This class can then be applied to multiple elements, and will affect all of them.
ids are similar to classes, but are declared with a #, and can only be applied to one element. ids also have precedence over classes in specificity, so ids will override class styling on elements.
